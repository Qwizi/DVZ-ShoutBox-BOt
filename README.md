# DVZ-ShoutBox-Bot
DVZ ShoutBox Bot
Version: 1.5.0

# General:
Bot sending messages on chat if user will register or write new thread/post. Responds to commands.

# Commands:
* `ban`
* `unban`
* `banlist`
* `prune`

# Create custom command:
* Create new file `YourCommand.php` in `inc/plugins/qwizi/Commands/` directory
